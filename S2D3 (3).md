## Ejercicio 1: El Inventario de la Tienda de Frutas

**El Problema:**
Tienes una lista de listas donde cada sublista representa un estante en una tienda de frutas. Cada número indica la cantidad de una fruta específica. Tu objetivo es calcular cuántas frutas útiles hay en total, pero con dos condiciones estrictas:
1.  Debe haber más de 5 unidades en ese espacio.
2.  La cantidad debe ser un número par.
```
# Lista de estantes con cantidades de productos
inventario = [
    [10, 4, 8],
    [5, 12, 3],
    [7, 20, 6]
]
```

### Conceptos Clave a Recordar:
* **Variable Acumuladora:** Es una variable que empieza en cero y se utiliza para ir guardando el resultado de sumas consecutivas dentro de un ciclo.
* **Operador Módulo (`%`):** Para saber si un número es par en programación, lo dividimos entre 2 y revisamos el residuo. Si el residuo es cero (`numero % 2 == 0`), significa que el número es par.
* **Operador Lógico `and`:** Obliga a que ambas condiciones se cumplan al mismo tiempo para que el código dentro del `if` se ejecute.

### Solución Explicada Paso a Paso

```python
# Lista de estantes con cantidades de productos (Nuestra Matriz)
inventario = [
    [10, 4, 8],  # Estante 0
    [5, 12, 3],  # Estante 1
    [7, 20, 6]   # Estante 2
]

# Paso 1: Declaramos una variable acumuladora para guardar la suma total.
# La inicializamos en 0 porque al inicio no hemos contado ninguna fruta.
total_frutas_validas = 0

# Paso 2: Recorremos la matriz principal (viajamos por cada estante).
for fila in inventario:
    
    # Paso 3: Recorremos los elementos de cada fila (revisamos cada caja del estante).
    for cantidad in fila: 
        
        # Paso 4: Aplicamos nuestro filtro doble.
        # Condición A: cantidad > 5
        # Condición B: cantidad % 2 == 0 (Verifica si el residuo de dividir entre 2 es cero, es decir, si es par)
        if cantidad > 5 and cantidad % 2 == 0:
            
            # Paso 5: Si cumple ambas condiciones, sumamos esa cantidad a nuestro total acumulado.
            # Escribir 'total_frutas_validas = total_frutas_validas + cantidad' es correcto,
            # pero en Python solemos usar el atajo '+=' que hace exactamente lo mismo.
            total_frutas_validas += cantidad

# Paso 6: Una vez que terminan los dos ciclos (terminamos de recorrer todo), imprimimos el resultado.
print("La cantidad total de frutas que cumplen las condiciones es:", total_frutas_validas)
```

## Ejercicio 2: Abarrotes y la Inflación

**El Problema:**
Eres el encargado de una tienda de abarrotes. Debido a la inflación, necesitas actualizar los precios en el sistema. La regla es clara: todos los productos que cuesten menos de 50 pesos deben subir un 10%. Los productos que cuesten 50 pesos o más no sufrirán cambios. Al finalizar, debes mostrar el sistema con los precios actualizados.

```
precios_pasillos = [
    [10.0, 55.0, 20.0],
    [45.0, 80.0, 12.0],
    [100.0, 30.0, 48.0]
]
```


### Conceptos Clave a Recordar:
* **Modificar vs. Leer:** A diferencia del Ejercicio 1 donde solo "leíamos" números para sumarlos, aquí necesitamos **cambiar o sobreescribir** los datos originales. Como vimos en la lección anterior, para modificar elementos dentro de una matriz de forma segura, **debemos usar sus índices o coordenadas** mediante `range(len())`.
* **Cálculo de Porcentajes:** Para aumentar un 10% a un valor, podemos multiplicarlo por `0.10` y sumárselo al valor original, o más directamente, multiplicarlo por `1.10`.



### pseudocódigo 
* declarar en una variable el 10 porciento de incremento 
* recorrer la matriz
* aplicar la condición (if)
    si el precio es menor a 50 
        aplicar el incremento, el cual es multiplicarlo por el 10 porciento 
        guardar/modificar el valor en la lista(matriz)

* mostrar la matriz ya modificada


### Solución Explicada Paso a Paso

```python
# Nuestra matriz inicial con los precios actuales
precios_pasillos = [
    [10.0, 55.0, 20.0], # Pasillo 0
    [45.0, 80.0, 12.0], # Pasillo 1
    [100.0, 30.0, 48.0] # Pasillo 2
]

# Paso 1: Definimos el factor de incremento para que el código sea fácil de entender y modificar a futuro.
# Multiplicar por 1.10 equivale a agregarle el 10% al valor original.
incremento = 1.10 

print("Precios antes de la inflación:")
for pasillo in precios_pasillos:
    print(pasillo)

print("\n--- Aplicando ajuste de precios ---\n")

# Paso 2: Recorremos la matriz usando las coordenadas (índices) porque necesitamos modificar los valores.
# len(precios_pasillos) nos da el número de filas.
for indice_fila in range(len(precios_pasillos)):
    
    # len(precios_pasillos[indice_fila]) nos da el número de columnas en esa fila específica.
    for indice_columna in range(len(precios_pasillos[indice_fila])):
        
        # Paso 3: Extraemos el precio actual usando sus coordenadas exactas para evaluarlo.
        precio_actual = precios_pasillos[indice_fila][indice_columna]
        
        # Paso 4: Aplicamos la condición. ¿El precio es menor a 50?
        if precio_actual < 50.0:
            
            # Paso 5: Calculamos el nuevo precio.
            nuevo_precio = precio_actual * incremento
            
            # Paso 6: Guardamos el nuevo precio en la misma posición (coordenada) de la matriz original,
            # sobreescribiendo el valor viejo. Usamos round(valor, 2) para dejar solo dos decimales.
            precios_pasillos[indice_fila][indice_columna] = round(nuevo_precio, 2)

# Paso 7: Mostramos la matriz ya modificada.
print("Precios actualizados después de la inflación:")
for pasillo in precios_pasillos:
    print(pasillo)
```
