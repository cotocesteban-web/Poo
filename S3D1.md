# funciones o métodos
## sintaxis:

## 1. ¿Qué es una Función? (La Analogía de la Receta)
Imaginen que una función es como una **receta de cocina**. 
* La receta tiene un **nombre** (ej. "Hacer Café").
* Tiene **pasos** específicos que siempre se siguen igual.
* Tú puedes "llamar" a esa receta cada vez que quieras café, sin tener que explicar paso a paso cómo encender la cafetera cada vez.

### Sintaxis Básica
Para crear una función usamos la palabra reservada `def` (de *define*). 

![](dia1-fun.png)

```python
def saludo():
    print("¡Hola! Bienvenido al sistema.")
    print("Es un gusto saludarte.")

# Llamada a la función (aquí es donde se ejecuta)
saludo()

```

## Ejemplo
```python
def saludo():
    print("hola buenos días!")

def despedida():
    print("adiós, que tengas un buen día!")


def menu():
    print("1. Saludar")
    print("2. Despedirse")
    print("3. Salir")

    opcion = input("Seleccione una opción: ")

    while True:
        match opcion:
            case "1":
                saludo()
            case "2":
                despedida()
            case "3":
                print("Saliendo...")
                break
            case _:
                print("Opción no válida")

        opcion = input("Seleccione una opción: ")


menu()
```


# parámetros en una función 
Parámetros: Enviando información a la función
A veces, nuestra "receta" necesita ingredientes específicos. En programación, estos ingredientes se llaman **parámetros**.

Si mi función es `saludo()`, pero quiero que sea personalizada, le paso el "ingrediente" `nombre`.

![](dia2-fun.png)

```python
def saludo(nombre):
    print("hola buenos días!")
    print(f"¿Cómo estás, {nombre}?")

def despedida(nombre):
    print("adiós, que tengas un buen día!")
    print(f"¡Hasta luego, {nombre}!")


def menu():
    print("1. Saludar")
    print("2. Despedirse")
    print("3. Salir")

    opcion = input("Seleccione una opción: ")

    nombre = input("Ingrese su nombre: ")

    while True:
        match opcion:
            case "1":
                saludo(nombre)
            case "2":
                despedida(nombre)
            case "3":
                print("Saliendo...")
                break
            case _:
                print("Opción no válida")

        opcion = input("Seleccione una opción: ")


menu()
```


# El Retorno de Valores: El comando `return`
Este es el concepto más importante. Imagina que envías a alguien a la tienda con dinero. Esa persona hace un proceso (comprar) y **regresa** con el pan. 

El `return` hace exactamente eso: termina el trabajo de la función y **devuelve** un resultado al programa principal para que podamos guardarlo en una variable.

> **Regla de Oro:** Una vez que Python lee un `return`, la función se detiene inmediatamente. Cualquier código escrito debajo de un `return` (dentro de la misma función) nunca se ejecutará.

## 2. función con parámetros


## 3. función con retorno 
Un supermercado desea calcular el precio final de una compra.
Cree una función llamada `calcular_total(precio, cantidad)` que:
* Reciba el precio de un producto.
* Reciba la cantidad comprada.
* Calcule el total a pagar.
* Devuelva el resultado al programa principal.

El programa debe mostrar el total final.

## 4. ejercicio integrador
Una sala de videojuegos vende fichas para poder jugar en las máquinas.
Cada ficha cuesta 500 colones.

El sistema debe permitir que varios clientes compren fichas durante el día.
El programa debe:

1. Mostrar un menú con las siguientes opciones:
```
1 - Comprar fichas
2 - Salir
```

2. Si el cliente desea comprar fichas:
* pedir el nombre del cliente
* pedir cuánto dinero tiene
* calcular cuántas fichas puede comprar
* calcular cuánto dinero le sobra

3. Reglas del sistema:
* si el cliente no tiene dinero suficiente para al menos una ficha, el sistema debe indicarlo
* si tiene suficiente dinero, se debe mostrar:
```
Cantidad de fichas que puede comprar
Dinero de vuelto
```

4. El programa debe repetirse usando un ciclo while hasta que el usuario elija salir.
5. El cálculo de fichas debe realizarse mediante una función con parámetros y retorno.
