# Repaso de Conceptos Básicos de Python

Antes de trabajar con condicionales, es importante recordar algunos conceptos fundamentales de programación en Python.

## Tipos de datos

Un **tipo de dato** define qué clase de información estamos almacenando en una variable.

Los tipos de datos más comunes en Python son:

| Tipo    | Descripción                     | Ejemplo                |
| ------- | ------------------------------- | ---------------------- |
| `int`   | Números enteros (sin decimales) | `10`, `-5`, `100`      |
| `float` | Números con decimales           | `3.14`, `2.5`, `100.0` |
| `str`   | Texto (cadena de caracteres)    | `"hola"`, `"Python"`   |
| `bool`  | Valores lógicos                 | `True`, `False`        |

### Ejemplo

```python
edad = 25        # int
precio = 19.99   # float
nombre = "Ana"   # str
activo = True    # bool
```

---

# Operadores Aritméticos

Los **operadores aritméticos** se utilizan para realizar operaciones matemáticas entre valores.

| Operador | Significado    | Ejemplo  | Resultado |
| -------- | -------------- | -------- | --------- |
| `+`      | Suma           | `5 + 3`  | `8`       |
| `-`      | Resta          | `10 - 4` | `6`       |
| `*`      | Multiplicación | `6 * 3`  | `18`      |
| `/`      | División       | `10 / 2` | `5.0`     |

### Ejemplo

```python
a = 10
b = 5

print(a + b)
print(a - b)
print(a * b)
print(a / b)
```

---

# Operaciones Especiales

Python también incluye algunos operadores matemáticos especiales.

| Operador | Nombre          | Descripción                                | Ejemplo   | Resultado |
| -------- | --------------- | ------------------------------------------ | --------- | --------- |
| `%`      | Módulo          | Devuelve el residuo de una división        | `10 % 3`  | `1`       |
| `//`     | División entera | Devuelve el resultado entero sin decimales | `10 // 3` | `3`       |

### Ejemplo

```python
print(10 % 3)   # residuo de la división
print(10 // 3)  # división entera
```

### ¿Para qué sirve el módulo `%`?

Se usa mucho para:

* saber si un número es **par o impar**
* ciclos repetitivos
* cálculos de tiempo

Ejemplo:

```python
numero = 8

if numero % 2 == 0:
    print("El número es par")
```

---

# Operadores Lógicos

Los **operadores lógicos** permiten combinar varias condiciones.

| Operador | Significado                           |
| -------- | ------------------------------------- |
| `and`    | ambas condiciones deben cumplirse     |
| `or`     | al menos una condición debe cumplirse |
| `not`    | invierte el valor lógico              |

### Ejemplo

```python
edad = 20
tiene_entrada = True

if edad >= 18 and tiene_entrada:
    print("Puede entrar")
```

### Ejemplo con `or`

```python
dia = "sabado"

if dia == "sabado" or dia == "domingo":
    print("Es fin de semana")
```

### Ejemplo con `not`

```python
llueve = False

if not llueve:
    print("Podemos salir")
```

---

# Operadores Comparativos

Los **operadores comparativos** comparan valores y siempre devuelven `True` o `False`.

| Operador | Significado   | Ejemplo  |
| -------- | ------------- | -------- |
| `>`      | mayor que     | `10 > 5` |
| `<`      | menor que     | `3 < 8`  |
| `==`     | igual a       | `5 == 5` |
| `>=`     | mayor o igual | `7 >= 7` |
| `<=`     | menor o igual | `3 <= 5` |
| `!=`     | diferente de  | `4 != 6` |

### Ejemplo

```python
edad = 18

print(edad >= 18)
```

---

# Casting (Conversión de tipos)

El **casting** permite convertir un tipo de dato a otro.

Esto es muy importante cuando usamos `input()`, ya que **todo lo que ingresa el usuario es texto (`str`)**.

| Función   | Convierte a    |
| --------- | -------------- |
| `int()`   | número entero  |
| `float()` | número decimal |
| `str()`   | texto          |

### Ejemplo

```python
edad = int(input("Ingrese su edad: "))
print(edad + 5)
```

Si no usamos `int()` ocurriría un error o concatenación de texto.

Ejemplo incorrecto:

```python
edad = input("Ingrese su edad: ")
print(edad + 5)
```

# Condicionales 
Los **condicionales** permiten que el programa tome decisiones.

El programa **evalúa una condición** y ejecuta cierto código dependiendo del resultado.

## La regla básica: el `if`

El `if` significa **"si se cumple esta condición entonces ejecuta este código"**.

### Sintaxis

```python
if condicion:
    accion
```

### Importante

Python usa **indentación (espacios o tabulación)** para definir los bloques de código.

Ejemplo conceptual:
```
bloque condicional del if 
    me indica que esto le pertenece al bloque
    esto tambien le pertenece

bloque condicional del if 
    me indica que esto le pertenece al bloque
    esto tambien le pertenece 
```

Todo lo que está **indentado** pertenece al bloque del `if`.

### Ejemplo
```python
dinero = 1000
precio = 750

if dinero >= precio:
    print("venta aprobada")
```
### Explicación

1. Se compara `dinero >= precio`
2. Si es **True**, se ejecuta el `print`
3. Si es **False**, el programa continúa sin ejecutar el bloque

# Condicional `else`

El `else` se usa para ejecutar código **cuando la condición del `if` NO se cumple**.

### Regla importante

`else` **NUNCA lleva condición**.

### Sintaxis

```python
if condicion:
    accion
else:
    accion alternativa
```

---

### Ejemplo

```python
dinero = float(input("dinero del cliente: "))
precio = 1200

if dinero >= precio:
    print("puede comprar el producto")
else:
    print("no puede comprar el producto")
```

### Explicación

Si el cliente tiene suficiente dinero → puede comprar.

Si no → se ejecuta el `else`.

# Condicional `elif`

`elif` significa **"else if"** (si no, pero si se cumple esta otra condición).

Se usa cuando tenemos **varias condiciones posibles**.

### Sintaxis

```python
if condicion1:
    accion
elif condicion2:
    accion
elif condicion3:
    accion
else:
    accion_final
```

---

# Ejemplo

```python
edad = int(input("Digite su edad: "))
precio_entrada = 2000 

if edad <= 12:
    print("el cliente es un niño, tiene un descuento del 50%")
    total = precio_entrada * 0.5

elif edad >= 60:
    print("el cliente es un adulto mayor, tiene un descuento del 25%")
    total = precio_entrada * 0.75

else:
    print("el cliente es un adulto, no tiene descuento")
    total = precio_entrada

print("El precio total de la entrada es: ", total)
```

### Explicación

1. Si edad ≤ 12 → descuento 50%
2. Si edad ≥ 60 → descuento 25%
3. En cualquier otro caso → precio normal

# Condicional: `match / case`

`match/case` es una estructura de **selección múltiple**.

Sirve para comparar un valor contra **muchas opciones posibles**.

Es similar al **switch** de otros lenguajes.

Disponible desde **Python 3.10**.

---

### Sintaxis

```python
match valor:
    case patron1:
        acciones
    case patron2:
        acciones
    case _:
        acciones por defecto
```

`case _` significa **cualquier otro caso** (equivalente a `else`).

---

# Ejemplo

```python
opcion = input("Seleccione (1-3): ")

match opcion:
    case "1":
        print("Registrando producto...")
    case "2":
        print("Mostrando inventario...")
    case "3":
        print("Saliendo...")
    case _:
        print("Opción inválida")
```

---

# Operadores dentro de `match`

Se pueden combinar opciones usando `|` que significa **OR**.

Ejemplo:

```python
case "1" | "registrar":
```

Esto significa:

```
opcion == "1" OR opcion == "registrar"
```

---

# Ejercicio más complejo

```python
print("Bienvenido al sistema de inventario")
print("1. registrar ")
print("2. mostrar")
print("3. salir")

opcion = input("Seleccione (1-3) o escriba salir ")
usuario = input("Ingrese su nombre de usuario: ")

match opcion:

    case ("registrar" | "1") if usuario == "admin":
        print("Registrando producto...")
        nombre = input("Ingrese el nombre del producto: ")

        if nombre == "pantalón":
            print("El producto ya existe")

        elif nombre == "camisa":
            print("El producto ya existe")  

        else:
            print("Producto registrado")

    case "mostrar" | "2":
        print("Mostrando inventario...")

    case "salir" | "3":
        print("Saliendo...")

    case _:
        print("Opción inválida")
```

---

# Explicación del ejemplo

Este programa combina varias cosas:

* `match/case`
* operadores lógicos
* condicionales anidados
* control de usuario

### Regla especial

```python
case ("registrar" | "1") if usuario == "admin":
```

Esto significa:

```
si opcion es registrar o 1
Y el usuario es admin
```

# ¿Cuándo usar `match/case` vs `if/elif/else`?

## Usar `match/case` cuando:

* hay **muchas opciones claras**
* se trabaja con **menús**
* se comparan **valores exactos**

Ejemplo típico:

```
menu de sistema
menu de videojuego
menu de inventario
```

---

## Usar `if/elif/else` cuando:

* se comparan **rangos numéricos**
* hay **condiciones complejas**
* se usan **operadores matemáticos**

Ejemplo típico:

```
edad
notas
temperatura
inventario
```

---
 **Regla práctica**

```
comparar valores exactos → match
comparar rangos o lógica → if
```
